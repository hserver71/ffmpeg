ffmpeg-devel -- FFmpeg development discussions and patches

 

About ffmpeg-devel
English (USA)

This list is about FFmpeg development discussions and patch submissions; but not for bug-reports. Please read the Code-of-conduct before posting, and refer to the developer documentation for info about submitting patches. Subscribing is not required to participate but is recommended.

You can read all list messages in the ffmpeg-devel Archives.

Using ffmpeg-devel

You must subscribe to the mailing list before posting a message.

Once you subscribe you can post a message to the mailing list by emailing your message to ffmpeg-devel@ffmpeg.org.

For more information on using the list see the FFmpeg Mailing List FAQ.

You can subscribe to the list, or change your existing subscription, in the sections below.
